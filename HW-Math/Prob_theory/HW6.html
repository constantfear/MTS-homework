Разобьм выборку на несколько сегментов:

  • Пол пользователя
  • Пользовался ли пользователь каким-либо официальным приложением магазина
  • Предпочитаемый язык пользователя

Для каждого разделения оценим:

  • Количество дней, пройденное с момента регистрации пользователя
  • Количество товаров, которые понравились пользователю
  • Число учетных записей пользователей, на которую подписан пользователь
  • Количество учетных записей, подписавшихся на действия пользователя

from scipy.stats import ttest_ind, f_oneway


# Попарные сравнения через тест Стьюдента
print("Сравнения с использованием теста Стьюдента:")
for df in other_df:
    stat, p_val = ttest_ind(df_en['seniority'], df['seniority'])
    print(f"p-value = {p_val}")

segments = [df_en['seniority'], df_fr['seniority'], df_de['seniority'], df_it['seniority'], df_es['seniority']]   
print("\nСравнения с использованием теста Фишера (ANOVA):")
stat, p_val = f_oneway(*segments)
print(f"p-value = {p_val}")

data = [1, 2, 3, 4, 5]
confidence_level = 0.95
mean = np.mean(df_male['seniority'])
std_dev = np.std(data, ddof=1)
sample_size = len(data)
margin_of_error = sts.t.ppf((1 + confidence_level) / 2, sample_size - 1) * (std_dev / np.sqrt(sample_size))
lower_bound = mean - margin_of_error
upper_bound = mean + margin_of_error

print(f"Точный доверительный интервал для среднего: ({lower_bound}, {upper_bound})")
